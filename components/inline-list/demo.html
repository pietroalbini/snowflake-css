Inline lists are just <ul>s displayed inline:

  • This is some text
  • And that's a link!
  • Hey
  • I'm in the second row!

They're useful for menus, footers, and other similar things. Sample code: 

<ul class="inline-list">
    <li>First row</li>
</ul>
<ul class="inline-list">
    <li>Second row</li>
</ul>
